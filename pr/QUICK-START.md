## To Add Videos to Your Portfolio:

### Step 1: Start the Server
```powershell
cd pr
npm run video-manager
```

### Step 2: Open Browser
Go to: **http://localhost:3000**

### Step 3: Add Your Video
1. Choose **Long-form** or **Short-form**
2. Enter video title
3. Paste Streamable URL
4. Click "Add Video to Portfolio"
5. **Done!** The video is automatically added and pushed to GitHub
